Using Default box-decoration-break as slice

This paragraph demonstrates the box-decoration-slice utility.
It keeps the background and border as one continuous shape.
Even if the text wraps across several lines,
the box remains unified without splitting,
creating a single clean highlight effect.

Using box-decoration-clone

This paragraph demonstrates the box-decoration-clone utility.
It applies a separate background and border to each line of wrapped text.
Instead of one continuous shape,
every line forms its own box,
creating a repeated highlight effect
that looks like individual tags or labels.
